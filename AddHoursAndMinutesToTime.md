Given: - a time (HH:MM) - hours to add (h_add) - minutes to add (m_add)

Add both to the time correctly and print the result.

Input Format

First line: time in HH:MM format
Second line: two integers h_add and m_add separated by space
Constraints

0 ≤ HH ≤ 23
0 ≤ MM ≤ 59
0 ≤ h_add ≤ 1000
0 ≤ m_add ≤ 10000
Output Format

Print the resulting time in 24-hour format (HH:MM).
